# hfjscli

A command-line interface for uploading and downloading files from Hugging Face Hub.

## Installation

```bash
npm install -g hfjscli
```

## Usage

### Upload files

```bash
hfjscli upload <repo-id> <file-path> [options]
```

### Download files

```bash
hfjscli download <repo-id> <file-path> [options]
```

## Development

### Setup

```bash
npm install
```

### Build

```bash
npm run build
```

### Test

```bash
npm test
```

### Development mode

```bash
npm run dev
```
